# MedAppoint – Online Medical Appointment Booking System

MedAppoint is a frontend-based web application that allows patients to book appointments with doctors online.

## Features

- User Registration (Doctor / Patient)
- Login & Logout System
- Role-Based Dashboard
- Doctor Listing
- Appointment Booking System
- Appointment Confirmation Modal
- Patient Appointment History
- Doctor Patient List
- Responsive UI Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for data simulation)

## Project Type

Frontend Web Application (No Backend)

## How It Works

The system stores users and appointments in the browser using LocalStorage.  
Doctors and patients have separate dashboards with different functionalities.
